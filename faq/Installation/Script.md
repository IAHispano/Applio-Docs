# Script
Here you will learn how to install Applio using commands in CMD.
## Windows
``` js
./run-install.bat
```
``` js
./run-applio.bat
```
## Linux/Mac
``` js
chmod +x run-install.sh
./run-install.sh
```
``` js
chmod +x run-applio.sh
./run-applio.sh
```
## Paperspace
``` js
make run-install
```
``` js
make run-applio
```
